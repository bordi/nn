# Mermaid Diagrams for `lab2`

У цій папці зібрано ключові `Mermaid`-діаграми для формального звіту й захисту:

- `01-system-pipeline.mermaid` — повний `RAG`-пайплайн з усіма основними компонентами;
- `02-retrieval-search-flow.mermaid` — як відбувається пошук `top-k` чанків;
- `03-evaluation-flow.mermaid` — як рахуються `Recall@K`, `MRR` і як формується manual answer evaluation;
- `04-artifacts-lifecycle.mermaid` — які артефакти створюються на кожному етапі.

Рекомендація для звіту:
- `System Pipeline` — для розділу архітектури;
- `Retrieval Search Flow` — для пояснення пошуку;
- `Evaluation Flow` — для розділу оцінювання;
- `Artifacts Lifecycle` — для розділу реалізації та відтворюваності.
