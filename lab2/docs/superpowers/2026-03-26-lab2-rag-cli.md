# План реалізації `lab2`: `RAG`-помічник у `CLI`

> **Для агентів-виконавців:** ОБОВ'ЯЗКОВА ПІДНАВИЧКА: використовуйте `superpowers:subagent-driven-development` (рекомендовано) або `superpowers:executing-plans` для поетапного виконання цього плану. Кроки оформлені через чекбокси (`- [ ]`) для відстеження прогресу.

**Мета:** Побудувати `lab2/` як інтерактивного `CLI` `RAG`-помічника на базі `SQuAD 2.0`, який відповідає на питання з цитатами й retrieval metadata та має базове оцінювання retrieval і якості відповідей.

**Архітектура:** Проєкт наслідує структуру `lab1`: одна CLI-точка входу `main.py`, невеликі модулі в `src/`, відтворювані артефакти в `artifacts/` і короткий `report.md`. Retrieval використовує локальні ембеддинги та `FAISS`, а генерація відповіді виконується через `OpenAI` із жорстким grounded prompt і відмовою від відповіді, якщо контекст недостатній.

**Технології:** Python, `sentence-transformers`, `faiss-cpu`, `openai`, `pandas`, `pytest`
